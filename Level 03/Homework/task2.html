About programming

Programming languages started with old mechanical computers and became modern tools. Early ones were hard to understand. The first programming language was developed in 1883 by Ada Lovelace, a mathematician.

Programming languages:

  • Python
  • JavaScript
  • C++
  • Java
  • C

Python

Created: 1991

Creator: Guido van Rossum

Type: high-level, general-purpose

Used for: web, AI, automation, data science

Popularity: very popular

JavaScript

Created: 1995

Creator: Brendan Eich

Type: high-level, scripting

Used for: websites, web apps, games

Popularity: very popular

C++

Created: 1985

Creator: Bjarne Stroustrup

Type: High-level, object-oriented

Used for: Games, engines, software

Popularity: Still widely used

Java

Created: 1995

Creator: James Gosling

Type: high-level, object-oriented

Used for: Android apps, web apps, software

Popularity: still widely used

C

Created: 1972

Creator: Dennis Ritchie

Type: high-level, procedural

Used for: operating systems, software

Popularity: still widely used
