ახსენით რისთვის გამოიყენება შემდეგი თეგები: nav, form, input, label და ატრიბუტები: type, id, for

nav გამოიყენება ნავიგაციის ბლოკის შესაქმნელად.
შეიცავს ბმულებს, რომლებიც მომხმარებელს საშუალებას აძლევს გადაადგილდეს საიტის სხვადასხვა სექციებზე.

form გამოიყენება ვებ-ფორმის შესაქმნელად, რომელიც მომხმარებელს საშუალებას აძლევს შეიყვანოს ინფორმაცია.
ფორმა შეიძლება შეიცავდეს შესატან ველებს, ღილაკებს და სხვა ელემენტებს.

input გამოიყენება შესატანი ველების შესაქმნელად, სადაც მომხმარებელი შეიყვანს ინფორმაციას.
type ატრიბუტის საშუალებით განისაზღვრება შესატანი ველის ტიპი (მაგ., ტექსტი, პაროლი, ელ.ფოსტა და ა.შ)

label გამოიყენება შესატანი ველის აღწერისთვის.
for ატრიბუტის საშუალებით დაკავშირებულია კონკრეტულ input ველთან.

type განსაზღვრავს input ველის ტიპს (მაგ., ტექსტი, პაროლი, ელ.ფოსტა, რადიო ღილაკი და ა.შ)

id უნიკალური იდენტიფიკატორი ელემენტისთვის, რომელიც გამოიყენება CSS ან JavaScript-ში ელემენტის ამოსაცნობად.

for გამოიყენება label ტეგში და მიუთითებს იმ ელემენტის id, რომელსაც აღწერს.
